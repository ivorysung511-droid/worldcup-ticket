# World Cup 2026 Ticket Marketplace

A complete ticket marketplace for FIFA World Cup 2026 built with Django.

## Features
- 🏆 Browse all World Cup 2026 matches
- 🔍 Search and filter events by stage, city, or team
- 💰 Compare ticket prices from multiple sellers
- 📱 Mobile-responsive design
- 🔒 Secure admin panel
